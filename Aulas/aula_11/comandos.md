# POO em TypeScript 

### Comandos usados na aula 
    -npm install -g typescript [Instalar o typescript]
    - tsc --init [inicializa as configurações do typescrip]
    - tsc --watch [transpila os arquivos ts para js]
    - rm [remove arquivos]
    - rm -rf [remove pastas]
